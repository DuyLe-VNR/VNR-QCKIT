## Ghi chú kỹ thuật

### IsRequireConsecutive binding issue
App dùng Kendo MVC + AngularJS — checkbox với hidden input companion. Khi click bằng `element.click()` JS thuần, Angular digest cycle không chạy → hidden input không sync. **Workaround cho spec.ts**: dùng Playwright `locator.check()` / `locator.uncheck()` thay vì `click()` — Playwright dispatch real pointer events kích hoạt Angular binding.

### UI không redirect/toast sau Save
Có thể app dùng inline notification (không phải toast pop-up) hoặc redirect chỉ xảy ra sau khi Kendo form validation pass. Cần investigate thêm khi có real user session.

### URL navigation
App dùng SPA hash routing: `https://host/#/Hrm_Main_Web/Controller/Action`. `window.location.href = '...'` hoạt động, `page.goto('/#/...')` không hoạt động với MCP browser. Trong spec.ts, BasePage.navigate() đã handle đúng.
